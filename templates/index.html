Chatter

A Pusher-powered chat application built using Flask

How to use:
- Open the Admin view: Admin View
- Start a chat in this window to chat with the admin
- Reply back from the admin window
- Open more instances of this window to add more client conversations

Enter a username and email in the chat window below. The username is displayed for the chat host, and the email serves as a unique identifier(it doesnt have to be a working address, just in the correct format)
